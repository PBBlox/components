## j-Stats12

- jComponent `v19|v20`

__Configuration__:

- `height {Number}` a height of chart (default: `120`)
- `max {Number}` a maximal hourly value (default: `undefined`)
- `border {Boolean}` enables border (default: `true`)
- `highlight {Boolean}` highlights the current hour (default: `false`)
