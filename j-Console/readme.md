## j-Console

- easy usage
- singleton
- modal
- supports app tabs
- supports __dark mode__

- jComponent `v19|v20`

__Configuration__:

- `datasource {String}` a path to data-source (learn from example)
- __NEW__ `clear {String}` clear logs, link to the `function(source)`

__Good to know__:

- `path` can be `{String}` with the `appid` due to focusing logs
