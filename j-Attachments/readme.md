## j-Attachments

The component renders attachments into the simple list. A user is able to remove attachments.

- jComponent `v19|v20`

A value be `Array` in the form:

```
[
	{
		name: 'attachment.docx',
		url: 'url_to_download',
		size: 1024, // in bytes
		dtcreated: Date
	}
]
```
