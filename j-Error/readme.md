## j-Error

- jComponent `v19|v20`
- easy usage
- errors are displayed only if the path contains an array in the form below:

```js
[
	{ error: 'Error Message 1' },
	{ error: 'Error Message 2' }
]
```

__Configuration__:
- `icon {String}` an error icon (default: `warning`)
