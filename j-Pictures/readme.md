## j-Pictures

The component displays images in a simple list. A user can remove and rearrange images.

- jComponent `v19|v20`
- it uses `j-Movable` component

__Configuration__:

- `grid {Number}` a column count in the grid (default: `4`)

---

A value be `Array` in the form:

```
[
	"/url_to_image_1.jpeg",
	"/url_to_image_2.jpeg",
	"/url_to_image_3.jpeg",
	"/url_to_image_4.jpeg"
]
```
