## j-EmptyInline

The component will display text inside the element if the value is empty or the array doesn't contain an item.

- jComponent `v19|v20`

__Configuration__:

- `grid {Number}` a column count in the grid (default: `4`)

---

A value be `Array` in the form:

```
[
	"/url_to_image_1.jpeg",
	"/url_to_image_2.jpeg",
	"/url_to_image_3.jpeg",
	"/url_to_image_4.jpeg"
]
```
