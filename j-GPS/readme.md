## j-GPS

- jComponent `v19|v20`
- singleton
- readonly

The component tries to read GPS coordinates.

__Configuration__:

- `watcher {Boolean}` enables a real-time watcher for GPS (default: `true`)

__Good to know__:

The component emits `ON('gps', function(coords) {` event with a new GPS coordinates.
