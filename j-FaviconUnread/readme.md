## j-FaviconUnread

- jComponent `v19|v20`

__Configuration__

- `fg {String}` foreground color (default: `white`)
- `bg {String}` backround color (default: `red`)
- `selector {String}` optional, selector for a specific image element (default: `favicon` in `head` of HTML)
